

# ファイル命名規則

各型やモジュールを記述するディレクトリ内では、以下の役割に応じたファイル名を用いて実装を分割します。

| ファイル名 | 役割・内容 |
| :--- | :--- | 
| `mod.rs` | モジュールの宣言および、他のファイルから公開する機能の再エクスポートを記述します。また、その型の独自の実装は基本的にここに記述します。 |
| `traits.rs` | そのモジュールやドメインで定義されるTraitを記述します。 |
| `impls.rs` | 対象となる型に対する、Traitの実装を記述します。|
| `constructor.rs` | 型の新規作成に関わる関数（`new`, `new_unchecked`）を記述します。 |
| `ops.rs` | 演算子に関する実装を記述します。 |
| `constants.rs` | モジュール内で使用される定数を記述します。|
| `random.rs` | ランダム生成に関わる実装を記述します。 |
