

# ファイル命名規則

まずは、`mod.rs`にすべてを書けば良いがコード量が多くなると見通しが悪くなるため下記のように分割することが望ましい。

| ファイル名 | 役割・内容 |
| :--- | :--- | 
| `mod.rs` | モジュールの宣言および、他のファイルから公開する機能の再エクスポートを記述する。また、その型の独自の実装は基本的にここに記述する。 |
| `traits.rs` | そのモジュールやドメインで定義されるTraitを記述する。 |
| `impls.rs` | 対象となる型に対する、Traitの実装を記述する。|
| `constructor.rs` | 型の新規作成に関わる関数（`new`, `new_unchecked`など）を記述する。 |
| `ops.rs` | 演算子に関する実装を記述する。 |
| `constants.rs` | モジュール内で使用される定数を記述する。|
| `random.rs` | ランダム生成に関わる実装を記述する。 |
